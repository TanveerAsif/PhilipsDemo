# PhilipsDemoRepo
 Opengl sample code to demonstrate 3d objects(Mesh) rendering, camera operation. Texture load  and Shader handling
 
1- Project Build in VS 2019 x64
2- In case of MSVSP110.dll missing, please installed redist package available in
   "Resource/Dependency" Folder.
3- To build choose x64 plateform of Debug/Release

4- Camera Key to Navigate
   Frwd / Backwrd / Left / Right
   Shift + Left / Right : Rotate left and right
   Control + Direction Keys : Speed up the navigation
